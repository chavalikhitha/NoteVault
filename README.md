# 📝 NoteVault - Notes Sharing Platform

## 📌 Project Description
NoteVault is a web-based application that allows users to create, manage, and share notes efficiently. It provides a simple and user-friendly interface for organizing notes digitally.

---

## 🚀 Features
- User Signup & Login
- Create, Edit, Delete Notes
- Search and Filter Notes
- Categorization (Study, Work, Personal)
- Share Notes with Other Users
- Dark Mode 🌙
- Responsive Design

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage (for data storage)
